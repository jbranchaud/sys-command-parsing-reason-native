# sys-command-parsing

> Execute a command on a forked process and read/parse the output

This is a tiny command line program written in [ReasonML](https://reasonml.github.io/) and compiled natively using `bsb-native`.

### Install

```
$ npm install
```

### Build

```
$ npm run build
```

### Run

```
$ lib/bs/bytecode/sys-command-parsing
```
